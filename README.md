# Currency Tracker API

Сервіс для моніторингу курсів валют через API Monobank.

## Технології
* **Django / DRF** (API)
* **PostgreSQL** (База даних)
* **Redis + Celery** (Фонові завдання)
* **Docker / Docker Compose** (Контейнеризація)
